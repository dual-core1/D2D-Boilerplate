# D2D Boilerplate
The tier below libraries like SDL. This is barely even an abstraction layer. If anything, it would go under an abstraction layer. The upside is that using only native libraries means there's very little overhead. The downside is this thing has never heard of portability.
## What's Inside?
Pretty much just the helper functions one would write when working on a D2D project. The APIs used here are:
- Win32
- Direct2D
- DirectInput
- XAudio2
- WIC
